Overview

Demo Purpose

This 7-minute showcase demonstrates the University Front Door Support Agent while setting up the AI Ethics session at 1:45 PM. The demo accomplishes two critical goals:

  1. Show the capability - Prove the technology works and is accessible
  2. Expose the ethical complexity - Create the "aha" moment that there are harder questions than "can we build it?"

Key Message

"Ethics Is The Edge"

The technology is democratized. The competitive advantage is in how you deploy it responsibly.

Slide 7: Three-Agent Architecture
4:05-5:35 (90s)
SCRIPT

[Click to advance slide, step toward screen, gesture to diagram]

"So how does it make those decisions? Three-agent orchestration.

[Point to QueryAgent]
